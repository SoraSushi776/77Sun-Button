## 语音补充

若您想为本按钮补充语音，您可以通过以下方式提交：

1. （推荐）前往 [issue](https://github.com/SoraSushi776/77Sun-Button/issues) 提交音频和音频描述（如这个音频的类型是什么，以及给这个音频取一个名字）

- 可爱

- 发狂
- 更多类型有待添加
